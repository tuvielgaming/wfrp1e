# WFRP 1e Foundry System

## Vision

Faithful implementation of Warhammer Fantasy Roleplay First Edition using the latest Foundry VTT architecture.

---

# Technical Goals

- Foundry VTT v14 only
- TypeScript
- Vite
- Rule Engine
- Active Effects
- UUID references
- Minimal duplicated data

---

# Design Principles

1. Actor stores only player state.
2. Compendiums are source of truth.
3. Rules calculate derived data.
4. Services modify documents.
5. Applications contain no business logic.
